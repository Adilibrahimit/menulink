# Calorie Reference Database

Standard calorie estimates for common Saudi restaurant menu items. Use as defaults when importing a new tenant's menu. Owner can adjust from `/admin/menu` (the nutrition panel).

Sources: USDA FoodData Central, Saudi SFDA nutritional guidelines, Foodics calorie database, and observed values from KO-KO + RzRz deployments.
